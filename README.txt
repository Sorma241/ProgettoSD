Il progetto progettoSD-Banca è stato realizzato in linguaggio java utilizzando il framework Spring. 
Il progetto contiene la classe BankApplication che si occupa di avviare il server web basato sulle REST API, le altre classi presenti 
servono per completare tutte le funzionalità richieste tra le quali, connessione al database locale usando SQLite tramite
libreria jdbc, creazione di tutti gli endpoint e Adattatore per le interfacce grafiche.
All'interno del progetto sono presenti anche i file html che descrivono le pagine web richieste, per il forntend sono 
stati usati i css forniti da bootstrap, il linguaggio html e javascript.
All'interno della cartella è presente il file BankDB.db che contiene il database.
